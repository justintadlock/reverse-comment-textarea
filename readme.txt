Moves the textarea back to the bottom of the comment form.

== Description ==

WordPress version 4.4 moves the textarea to the top of the comment form.  This plugin moves the textarea back to the bottom to what it was like pre-4.4.

== Installation ==

1. Upload `reverse-comment-textarea` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Settings > Reverse Comment Textarea" to select which settings you'd like to use.

== Frequently Asked Questions ==

### Why was this plugin created?

WordPress 4.4 puts the textarea at the top of the comment form.  This plugin was created for users who want the old, pre-4.4 style with the textarea at the bottom.

### How do I use it?

Just install and activate.  Nothing else to do.
